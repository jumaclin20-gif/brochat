# 💪 Bro Chat

> Real-time fitness motivation + world time + gun game for bros in Kisumu & USA

**Live App:** `https://jumaclin20-gif.github.io/brochat/`

### ✨ Features

| Feature | Details |
| --- | --- |
| **💭 Daily Motivation** | 8+ gym quotes. Click "New Quote" |
| **🕐 Live World Time** | Kisumu + USA ET. Updates every second |
| **💵 USD→KES Rate** | Live exchange rate. Auto-updates every 5min |
| **🔫 Gun Simulator** | Score game with 30 ammo. Vibration on fire |
| **💬 Local Notes** | Save notes offline. Data stays on your phone |
| **📱 Mobile First** | Works 100% on Android Chrome. No install |

### 🚀 How It Works 

1. **No backend needed** - Runs entirely in your browser
2. **PWA Ready** - Add to home screen for app-like experience  
3. **Offline First** - Time + game + notes work without internet
4. **Exchange rate** uses free `er-api.com` - cached if offline

### 📲 Install as App

1. Open the live link on Android Chrome
